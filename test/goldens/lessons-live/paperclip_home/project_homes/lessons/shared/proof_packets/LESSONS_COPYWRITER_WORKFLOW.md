# Lessons Copywriter Workflow

This file defines the shared workflow for the Lessons Copywriter lane.

It does not define handoff mechanics, the shared packet file rules, or
critic verdicts.

## What This Lane Must Do

- read the whole lesson before rewriting isolated strings
- inventory every learner copy area that changed materially
- confirm current `lesson_root/_authoring/ACTION_AUTHORITY.md` coverage before
  rewriting an exact `you should X here` line
- use `lessons_content_standards/LESSONS_POKER_GROUNDING_RULES.md` for material learner copy
- keep visible action button labels on the contract owned by
  `lessons_content_standards/LESSON_COPY_STANDARDS.md`
- keep feedback aligned with the current answer contract
- revalidate the manifest after the copy pass

## Files That Count As The Packet

Name the copy packet exactly:

- `lesson_root/lesson_manifest.json`
- `lesson_root/COPY_RECEIPTS.md`
- `lesson_root/ANTI_LEAK_AUDIT.md`
- refreshed validation files cited as live proof
- the current run gate bundle and named files when a run gate bundle
  applies
